# 第一章：理解图式——你为什么是现在这样

## 什么是图式

图式不是性格缺陷，不是"想太多"，不是"不够坚强"。

图式是**童年期为适应不安全环境而发展出的生存策略**。它在当时是聪明的、必要的、保护性的。问题在于：环境变了，策略没跟着更新。

打个比方：你小时候住在一个随时可能失火的房子里，你学会了永远保持警惕、不脱衣服睡觉、把重要东西打包好随时能跑。这个策略在那个房子里完全合理。

现在你长大了，搬进了一个安全的房子。但你还在打包、还在和衣而睡、还在听每一个响动。不是因为你傻，是因为**你的身体还没有更新"安全"这个判断**。

图式改变不是"想通了就没事了"。它需要三个层面同时工作：
1. **认知层**：理解图式的逻辑（头脑层面）
2. **体验层**：在情感层面经历新的体验（身体/情绪层面）
3. **行为层**：用新的行为模式替代旧的（行动层面）

只做认知分析 = 你理解了但改变不了。
只做行为改变 = 你坚持不了因为内在冲突没解决。
三个层面必须协同。

# 第二章：情感抑制图式——你的核心防御

## 得分：5.33/7（全场最高）

### 2.1 它是怎么形成的

情感抑制图式的典型形成环境：

- **情绪表达被惩罚或忽视**："男孩子哭什么哭"、"别那么敏感"、"这有什么好生气的"
- **情绪表达带来负面后果**：表达情绪后被嘲笑、被冷暴力、被利用
- **家庭中情绪是危险的**：父母情绪不稳定，你学会了"不添乱"
- **过度奖励理性**：从小因为"懂事"、"成熟"、"不让人操心"而被表扬
- **文化环境**：男性气质规范（"男人要坚强"）、东亚文化中的情感克制

**你需要回忆的**：在你的成长经历中，什么时候你学会了"情绪是危险的/没用的/不被接受的"？

不是要你去怪父母或怪环境。是去**理解**这个策略是怎么被安装进来的。

### 2.2 它的运行机制（维持循环）

触发事件（有人让你失望/你感到脆弱/关系变亲密）→ 身体信号出现（胸口发紧/喉咙堵/眼眶热/心跳加速）→ 图式激活警报："危险！不能失控！" → 防御策略启动：理智化/转移注意力/合理化/物理隔离 → 情绪被压下去 → 短期效果："控制住了" → 长期代价：情绪积累、情感剥夺加重、亲密关系停留表面、图式固化

### 2.3 它在你生活中的具体表现

**在亲密关系中：**
- 分手后你不是不痛，而是用理性分析把痛隔离了
- 伴侣说"你不够亲近"或"你像一堵墙"
- 你很难说出"我需要你"、"我很害怕"、"我很想你"

**在日常生活中：**
- 你用"分析问题"来替代"感受问题"
- 你可能有某些身体症状：失眠、胃部不适、头痛、肌肉紧张
- 你可能觉得自己"没什么感觉"——但没感觉本身就是一种感觉，是麻木

**在认知层面：**
- 你觉得"理性分析"比"感受情绪"更高级
- 你觉得如果自己也情绪化了，就"失控"了、"不行了"

### 2.4 改变策略

**策略A：身体觉察训练（每天5分钟）**
情感抑制的人和情绪之间的连接是断的。重建连接从身体开始。
练习：闭眼，从头顶到脚底逐步感受身体，找到感觉后待在那里感受30秒。

**策略B：情绪命名练习（每天1次）**
每天晚上：今天我感受到了______，强度是____，发生在______

**策略C：渐进式情绪暴露（每周1次）**
第1周：日记里写一句真实感受 → 第2周：对朋友说"我最近不太好" → 第3周：表达小型不满 → 第4周：表达正面情感 → 第5周：脆弱时不离开 → 第6周：说"我害怕"

**策略D：认知解构（ACT技术）**
"我注意到我现在在分析这件事。我注意到我正在用思考来避免感受。"

# 第三章：断连与拒绝图式域

## 整体得分：4.17/7（中偏高）

核心信念：**"我不够好，所以别人会离开我。即使不离开，也不会真正理解我。"**

### 3.1 被遗弃/不稳定图式（4.33）

核心信念：我爱的人最终都会离开。
表现：关系稳定时反而不安、对方冷淡就灾难化、可能在被离开之前先离开。

### 3.2 情感剥夺图式（4.33）

核心信念：没有人会真正理解我、满足我的情感需求。
三种形式：滋养剥夺、共情剥夺、保护剥夺。

致命循环：情感抑制 → 不表达需求 → 对方不知道你需要什么 → 情感剥夺感加重 → 更不敢表达

### 3.3 不信任/虐待图式（4.0）

核心信念：别人会利用我、伤害我、背叛我。
这是回避依恋的认知内核。

### 3.4 缺陷感/羞耻图式（4.0）

核心信念：我内在有什么根本性的问题，如果被看到就会被拒绝。
缺陷感 + 苛刻标准 = "我有缺陷，所以我必须完美来补偿。"

### 3.5 改变策略

**策略E：识别"旧剧本"vs"当下现实"**
触发事件 → 旧剧本 → 证据 → 替代解读

**策略F：矫正性情感体验**
做微小的信任测试，观察"旧剧本是否成真"

**策略G：内在安全基地建设**
想象一个完全安全的地方，一个理想的照顾者对你说："你不需要完美才值得被爱。"

# 第四章：苛刻标准图式

## 得分：4.67/7

核心信念："如果我不做到最好，就没有价值。"

形成：父母的爱有条件，只有成就被认可。
表现：拖延不是因为懒而是怕做不好、完成一件事只看到不足、自我价值外部锚定。

### 改变策略

**策略H：区分健康高标准 vs 苛刻标准**
"我想做好" vs "我必须做到最好"

**策略I：刻意不完美实验**
第1周：交70分的作业 → 第2周：发没编辑的消息 → 第3周：说"我不确定" → 第4周：做怕做不好的事

**策略J：内在价值锚定**
每天写下："今天，我不需要做任何事来证明我的价值。我作为一个人，本身就有价值。"

# 第五章：整合——改变地图

## 改变阶段

| 阶段 | 时间 | 内容 |
|------|------|------|
| 觉察期 | 现在 | 理解图式模式 |
| 探索期 | 第1-4周 | 身体觉察和情绪命名 |
| 实验期 | 第2-8周 | 渐进暴露和刻意不完美 |
| 整合期 | 第3-6月 | 新模式开始自动化 |
| 巩固期 | 第6-12月 | 处理深层图式 |

## 每日练习

- **晨间（3分钟）**：今天可能遇到什么图式触发？
- **日间（随时）**：情绪被压下时暂停30秒感受
- **晚间（5分钟）**：身体扫描 + 情绪温度计 + 图式觉察
- **每周（30分钟）**：回顾本周模式

## 何时需要专业帮助

- 情绪练习时出现强烈恐慌或解离
- 渐进暴露反复失败
- 用更极端方式压抑情绪
- 睡眠/食欲/日常功能受影响
- 持续感到绝望或无意义感

# 第六章：当前阶段特别建议

## 6.1 关于分手
分手可能还没有被真正处理。情感抑制让你"分析通了"，但情绪层面可能还压着。

## 6.2 关于找人生支柱
真正的身份探索不是通过思考完成的，而是通过体验。

## 6.3 最大的风险
**用"自我分析"替代"自我改变"**。如果你做完评估后只是"理解了"，但生活中没有任何行为上的改变，那情感抑制又一次成功地把你隔离了。

**最小的行动**：今晚就做一次情绪温度计。"今天我感受到了______。"
