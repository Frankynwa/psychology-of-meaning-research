# 第一章：理解图式——你为什么是现在这样

## 什么是图式

图式不是性格缺陷，不是"想太多"，不是"不够坚强"。

图式是**童年期为适应不安全环境而发展出的生存策略**。它在当时是聪明的、必要的、保护性的。问题在于：环境变了，策略没跟着更新。

打个比方：你小时候住在一个随时可能失火的房子里，你学会了永远保持警惕、不脱衣服睡觉、把重要东西打包好随时能跑。这个策略在那个房子里完全合理。

现在你长大了，搬进了一个安全的房子。但你还在打包、还在和衣而睡、还在听每一个响动。不是因为你傻，是因为**你的身体还没有更新"安全"这个判断**。

图式改变不是"想通了就没事了"。它需要三个层面同时工作：
1. **认知层**：理解图式的逻辑（头脑层面）
2. **体验层**：在情感层面经历新的体验（身体/情绪层面）
3. **行为层**：用新的行为模式替代旧的（行动层面）

只做认知分析 = 你理解了但改变不了。
只做行为改变 = 你坚持不了因为内在冲突没解决。
三个层面必须协同。

# 第二章：情感抑制图式——你的核心防御

## 得分：5.33/7（全场最高）

### 2.1 它是怎么形成的

情感抑制图式的典型形成环境：

- **情绪表达被惩罚或忽视**："男孩子哭什么哭"、"别那么敏感"、"这有什么好生气的"
- **情绪表达带来负面后果**：表达情绪后被嘲笑、被冷暴力、被利用
- **家庭中情绪是危险的**：父母情绪不稳定，你学会了"不添乱"
- **过度奖励理性**：从小因为"懂事"、"成熟"、"不让人操心"而被表扬
- **文化环境**：男性气质规范（"男人要坚强"）、东亚文化中的情感克制

**你需要回忆的**：在你的成长经历中，什么时候你学会了"情绪是危险的/没用的/不被接受的"？

不是要你去怪父母或怪环境。是去**理解**这个策略是怎么被安装进来的。

### 2.2 它的运行机制（维持循环）

```
触发事件（有人让你失望/你感到脆弱/关系变亲密）
    ↓
身体信号出现（胸口发紧/喉咙堵/眼眶热/心跳加速）
    ↓
图式激活警报："危险！不能失控！"
    ↓
防御策略启动：
    ├─ 理智化："从逻辑上分析一下这件事……"
    ├─ 转移注意力："想点别的/打游戏/工作"
    ├─ 合理化："其实这没什么大不了"
    └─ 物理隔离：离开现场/不回消息/独处
    ↓
情绪被压下去了
    ↓
短期效果：感觉"控制住了"，松了口气
    ↓
长期代价：
    ├─ 情绪积累，最终以爆发或身体症状释放
    ├─ 没人了解真实的你 → 情感剥夺感加重
    ├─ 亲密关系停留在表面 → 被遗弃恐惧持续
    └─ 你觉得"我就是这样" → 图式进一步固化
```

### 2.3 它在你生活中的具体表现

根据你的评估数据，以下是你可能出现的模式：

**在亲密关系中：**
- 分手后你不是不痛，而是用理性分析把痛隔离了
- 你可能很快就能"想通"分手这件事，但身体/情绪反应被压在下面
- 伴侣说"你不够亲近"或"你像一堵墙"
- 你很难说出"我需要你"、"我很害怕"、"我很想你"
- 当别人向你表达情感时，你不知道怎么回应

**在日常生活中：**
- 你用"分析问题"来替代"感受问题"
- 你可能在独处时偶尔有莫名的情绪涌上来，但很快压下去
- 你可能有某些身体症状：失眠、胃部不适、头痛、肌肉紧张（情绪找不到出口就会从身体出来）
- 你习惯性地"看开"事情，但内心深处并不平静
- 你可能觉得自己"没什么感觉"——但没感觉本身就是一种感觉，是麻木

**在认知层面：**
- 你觉得"理性分析"比"感受情绪"更高级
- 你对别人"情绪化"可能有轻微的不屑
- 你觉得如果自己也情绪化了，就"失控"了、"不行了"
- 你把"我能处理好情绪"等同于"我不需要有情绪"

### 2.4 情感抑制的真实代价（你以为的vs实际的）

| 你以为 | 实际 |
|--------|------|
| 我控制住了情绪 | 你控制住了情绪的**表达**，情绪本身并没有消失 |
| 我很理性 | 你在用理性**防御**感受，不是真正地理性 |
| 我不需要别人理解 | 你需要，但你已经放弃了被理解的希望 |
| 分手我想通了 | 你**分析通了**，但情绪可能还压在身体里 |
| 我没什么感觉 | 你有，但你已经切断了感受通道 |

### 2.5 改变策略

#### 策略A：身体觉察训练（每天5分钟）

情感抑制的人和情绪之间的连接是断的。你不是没有情绪，你是**感觉不到**。重建连接从身体开始。

**练习：身体扫描**
1. 每天找一个安静的5分钟
2. 闭眼，从头顶到脚底逐步感受身体
3. 问自己："我身体哪个部位有感觉？"
4. 不需要命名情绪，只需要描述身体感觉："胸口有点紧"、"肩膀在发酸"、"腹部有东西在动"
5. 当你找到一个感觉，不要分析它，只是**待在那里感受它30秒**

**为什么这有效**：情绪永远先在身体出现，然后才被大脑命名。你切断的是大脑那一层，但身体还在记录。重新连接身体就是重新连接情绪的第一步。

#### 策略B：情绪命名练习（每天1次）

**练习：情绪温度计**
每天晚上花2分钟，完成这个句子：

> "今天我感受到了______（情绪词），强度是____（1-10），发生在______（场景）"

**情绪词库**（从最基础的开始，不要一上来就用"存在性焦虑"这种词）：

| 基础层 | 进阶层 | 高级层 |
|--------|--------|--------|
| 开心 | 满足 | 感恩 |
| 难过 | 失落 | 悲伤 |
| 生气 | 沮丧 | 愤怒 |
| 害怕 | 不安 | 恐惧 |
| 讨厌 | 不舒服 | 厌恶 |
| 惊讶 | 意外 | 震惊 |
| — | 孤独 | 被遗弃感 |
| — | 羞耻 | 无价值感 |
| — | 嫉妒 | 被剥夺感 |
| — | 矛盾 | 困惑 |

**关键规则**：不要评判你的情绪。"我不应该感到嫉妒"这种想法本身就是情感抑制在工作。

#### 策略C：渐进式情绪暴露（每周1次）

这是最核心也最难的策略。你不需要一步到位"彻底敞开"。你只需要**每个月比上个月多表达一点点**。

**阶梯设计：**

| 阶段 | 行动 | 难度 |
|------|------|------|
| 第1周 | 在情绪日记里写下一句真实的感受（只有你自己看） | ⭐ |
| 第2周 | 对一个安全的朋友说一句"我最近状态不太好" | ⭐⭐ |
| 第3周 | 在关系中表达一个小型不满（"你那样说让我不太舒服"） | ⭐⭐⭐ |
| 第4周 | 表达一个正面情感（"和你在一起我很开心"） | ⭐⭐⭐ |
| 第5周 | 在感到脆弱时，不离开，留在原地 | ⭐⭐⭐⭐ |
| 第6周 | 说出"我害怕"或"我需要你" | ⭐⭐⭐⭐⭐ |

**注意**：每一步都可以重复多周，直到你觉得"还行，没那么可怕了"再进入下一步。不要赶进度。

#### 策略D：认知解构（ACT技术）

当你发现自己在用理智化防御时，练习这个句式：

> "我注意到我现在在分析这件事。我注意到我正在用思考来避免感受。"

不是要你停止分析。只是**注意到**你在做这件事。这个"注意到"本身就创造了一个空间——你不是你的防御策略，你是那个**看到**防御策略的人。

这就是ACT说的**"自我即情境"（self-as-context）**：你不是情感抑制，你是那个**有**情感抑制这个模式的人。

# 第三章：断连与拒绝图式域——你为什么总在关系里觉得不够

## 整体得分：4.17/7（中偏高）

这个域有4个子图式，它们共同构成了一个核心信念：

> **"我不够好，所以别人会离开我。即使不离开，也不会真正理解我。"**

### 3.1 被遗弃/不稳定图式（4.33）

**核心信念**：我爱的人最终都会离开。

**它是怎么形成的：**
- 早期分离经历（父母不在身边、频繁搬家、被寄养）
- 照顾者的不可预测性（有时在有时不在，情感上忽冷忽热）
- 失去重要关系的经历
- 父母本身有焦虑依恋，把"世界是不安全的"传递给了你

**它在你生活中的表现：**
- 关系稳定时你反而不安——"太好了所以不会持久"
- 对方稍微冷淡一点你就开始灾难化："TA要离开了"
- 你可能在对方离开你**之前**先离开对方（主动结束来控制局面）
- 分手后最痛的不是失去对方，而是"果然如此"的确认感——"我早就知道会这样"

**它和情感抑制的关系：**
被遗弃恐惧被情感抑制压住了。你表面上看起来"分手后恢复得挺快"，但底层的被遗弃恐惧并没有被处理，只是被封存了。下一段关系它会原封不动地冒出来。

### 3.2 情感剥夺图式（4.33）

**核心信念**：没有人会真正理解我、满足我的情感需求。

**三种形式（你可能同时有几种）：**
1. **滋养剥夺**：没有人给我温暖、关心、安慰
2. **共情剥夺**：没有人真正理解我的感受
3. **保护剥夺**：没有人在我需要时站在我这边

**它在你生活中的表现：**
- 你很少向别人提情感需求，因为"提了也没用"
- 当别人关心你时，你觉得不自在或不相信是真心的
- 你可能觉得自己在关系中总是"给予者"而不是"接受者"
- 你内心有一个很深的孤独感，但你已经习惯了

**它和情感抑制的致命循环：**

```
情感抑制："不能表达情绪"
    ↓
你在关系中不表达情感需求
    ↓
对方不知道你需要什么 → 对方没有满足你
    ↓
情感剥夺感加重："果然没有人理解我"
    ↓
你更不敢表达了："说了也没用"
    ↓
循环加深
```

**这是一个自我实现的预言。** 你不是不值得被理解，而是你从来没有给过别人理解你的机会。

### 3.3 不信任/虐待图式（4.0）

**核心信念**：别人会利用我、伤害我、背叛我。

**它在你生活中的表现：**
- 你对别人的动机有本能的怀疑
- 你需要很长时间才能信任一个人
- 当别人对你好时，你第一反应是"TA想从我这得到什么"
- 你在关系中会保留信息、保留真实想法

**它和回避依恋的关系：**
不信任图式是回避依恋的**认知内核**。你不靠近别人不是因为你不需要，而是因为你靠近后被伤害的风险太大。

### 3.4 缺陷感/羞耻图式（4.0）

**核心信念**：我内在有什么根本性的问题，如果被看到就会被拒绝。

**它在你生活中的表现：**
- 你害怕被人"真正了解"后会被嫌弃
- 你有某些关于自己的秘密，觉得绝对不能让人知道
- 当别人批评你时，你不是愤怒，而是羞耻——"TA说得对，我就是这样"
- 你可能在社交中表面自信，但内心觉得自己是"冒牌货"（imposter）

**它和苛刻标准的关系：**
缺陷感 + 苛刻标准 = "我有缺陷，所以我必须完美来补偿。一旦不完美，缺陷就暴露了。"

### 3.5 断连域的改变策略

#### 策略E：识别"旧剧本"vs"当下现实"

当你感到被遗弃/被剥夺/不被信任/有缺陷感时，做这个练习：

1. **写下触发事件**（发生了什么）
2. **写下你的自动解读**（你的图式告诉你的版本）
3. **写下证据**（支持这个解读的客观事实是什么？反对的呢？）
4. **写下替代解读**（一个没有图式滤镜的人会怎么看这件事？）

**示例：**

| 触发事件 | 旧剧本 | 证据 | 替代解读 |
|----------|--------|------|----------|
| 对方2小时没回消息 | "TA不在乎我了" | TA以前也有忙的时候 | TA可能在忙，上次也没回消息但后来解释了 |
| 朋友取消了约定 | "果然靠不住" | 这是第一次取消 | TA生病了，不是故意的 |
| 分手 | "我永远会被抛弃" | 这段关系确实结束了 | 这段关系结束了，不代表所有关系都会结束 |

**注意**：这不是"正能量洗脑"。你不需要相信替代解读。你只需要**看到旧剧本不是唯一可能的解读**。

#### 策略F：建立"矫正性情感体验"

图式是通过**重复的负面体验**形成的。改变它需要**重复的正面体验**——不是一次顿悟，而是反复积累。

**具体做法：**

1. **识别安全的人**：在你的生活中，有没有一个人是相对安全的？（不一定是最好的朋友，可能只是一个不会评判你的同学或同事）

2. **做微小的信任测试**：
   - 第1步：分享一个不重要的个人信息（"我其实不太喜欢吃XX"）
   - 第2步：分享一个中等脆弱的信息（"我最近压力挺大的"）
   - 第3步：分享一个真实感受（"那件事让我挺难过的"）
   - 观察：对方的反应是否如你图式预测的那样（评判/离开/利用）？

3. **记录结果**：大多数情况下，对方的反应会比你图式预测的好。把每一次"旧剧本没有成真"的记录下来。这不是为了否认你的感受，而是为了**给大脑提供新的数据**。

#### 策略G：内在安全基地建设（依恋理论 × 图式疗法）

你没有一个内在的"安全基地"可以依靠。你需要建立一个。

**练习：安全基地意象**
1. 闭上眼睛
2. 想象一个让你感到完全安全的地方（真实的或想象的）
3. 在这个地方里，有一个"理想的照顾者"——不是你真实的父母，而是**你需要的那种**存在
4. 想象这个存在对你说：**"你不需要完美才值得被爱。你不需要隐藏自己才安全。你可以留下来。"**
5. 注意你身体的反应。如果你觉得荒谬、想哭、或者什么感觉都没有——都是正常的

**这个练习的神经科学基础**：大脑的依恋系统无法区分"真实的安全关系"和"生动想象的安全关系"。反复练习安全意象可以逐步降低依恋系统的警报敏感度。

# 第四章：苛刻标准图式——你永远不够好

## 得分：4.67/7

### 4.1 核心信念

> "如果我不做到最好，就没有价值。"

### 4.2 它是怎么形成的

- 父母的爱有条件："考得好→表扬，考得差→失望/冷暴力"
- 家庭中只有"成就"被认可，其他（感受、存在本身）不被看见
- 早期的高期望被内化为自我标准
- 你可能从小就是"别人家的孩子"——但代价是你觉得自己的价值等于你的表现

### 4.3 它在你生活中的表现

**在学业/工作：**
- 你很难接受"够好就行"——总觉得可以再好一点
- 拖延不是因为懒，是因为"如果我做不出最好的，不如不开始"
- 完成一件事后你不会庆祝，只会看到不足的部分
- 你对自己的要求远高于对别人的要求

**在关系中：**
- 你觉得自己必须"足够好"才配被爱
- 你在关系中可能过度付出（用行为弥补"内在不够"）
- 你害怕暴露缺点，因为缺点 = 不完美 = 不值得被爱
- 你在关系中可能有"冒牌货"感

**在自我认知中：**
- 你的自我价值是**外部锚定**的：成绩、能力、别人的认可
- 你没有一个稳定的"我本身就够好"的内在感觉
- 你可能在某个领域很强，但内心始终觉得"还不够"

### 4.4 苛刻标准 × 情感抑制 的致命组合

这两个图式联手制造了一个陷阱：

```
苛刻标准："我必须完美"
    ↓
你做到了很多，但永远觉得不够
    ↓
焦虑/压力/挫败感出现
    ↓
情感抑制："不能表现出来，失控是不完美的"
    ↓
你把压力压下去，继续追求完美
    ↓
情绪积累 → 身体症状/间歇性崩溃/麻木
    ↓
你把崩溃也当成"不完美" → 更苛刻地要求自己
    ↓
循环加深
```

### 4.5 改变策略

#### 策略H：区分"健康的高标准"和"苛刻标准"

| 健康的高标准 | 苛刻标准 |
|-------------|---------|
| "我想做好" | "我必须做到最好" |
| 做到了→满意 | 做到了→还不够好 |
| 失败了→失望但能接受 | 失败了→自我否定 |
| 标准来自内在选择 | 标准来自恐惧（不完美=没价值） |
| 可以放松 | 放松=偷懒=有罪 |

**练习**：下次你对自己不满时，问自己：
- "如果我的朋友做了同样的事，我会怎么评价TA？"
- 如果你对朋友的标准比对自己的标准低，那就是苛刻标准在工作。

#### 策略I：刻意不完美实验（行为层面）

这是图式疗法中的"行为模式对抗"——通过刻意做出不符合图式的行为来削弱图式。

**实验设计：**

| 周次 | 实验 | 目的 |
|------|------|------|
| 第1周 | 交一份"70分"的作业，不修改到"完美" | 体验"不够好"不会真的导致灾难 |
| 第2周 | 在聊天中发一条没反复编辑的消息 | 体验不完美的表达不会被拒绝 |
| 第3周 | 对别人说"我不确定"或"我不会" | 体验暴露弱点不会导致被抛弃 |
| 第4周 | 做一件你一直想做但怕做不好的事 | 体验过程比结果重要 |

**每次实验后记录：**
1. 我预期会发生什么？（图式的预测）
2. 实际发生了什么？
3. 我的图式预测准确吗？

**大多数情况下你会发现：图式预测的灾难并没有发生。**

#### 策略J：内在价值锚定

你的自我价值不应该等于你的表现。你需要一个不依赖于"做了什么"的"我够好"的感觉。

**练习：存在 vs 做**

每天写下：
> "今天，我不需要做任何事来证明我的价值。我作为一个人，本身就有价值。"

你第一次读这句话时可能觉得空洞或不真实。这很正常。这不是要你马上相信，而是要你的大脑开始**接触**这个可能性。

# 第五章：整合——你的改变地图

## 5.1 改变的阶段

| 阶段 | 时间 | 内容 | 标志 |
|------|------|------|------|
| **觉察期** | 现在 | 理解自己的图式模式 | 你能说出"这是我的情感抑制在工作" |
| **探索期** | 第1-4周 | 开始练习身体觉察和情绪命名 | 你开始能感觉到情绪在身体中的存在 |
| **实验期** | 第2-8周 | 做渐进式暴露和刻意不完美实验 | 你发现"旧剧本没有成真"的次数在增加 |
| **整合期** | 第3-6月 | 新的模式开始自动化 | 你不再需要刻意提醒自己，新反应开始自然出现 |
| **巩固期** | 第6-12月 | 处理深层图式，建立稳定的内在安全基地 | 你能在一个安全的关系中表达脆弱 |

**注意**：这不是线性的。你会反复、会倒退、会觉得"白练了"。这些都是正常的。图式改变更像螺旋上升而不是直线前进。

## 5.2 每日练习计划

### 晨间（3分钟）
> "今天我可能会遇到的图式触发是什么？如果触发了，我可以用什么替代反应？"

### 日间（随时）
> 当你注意到情绪被压下去时（身体信号：胸口紧/喉咙堵/肩膀发硬），暂停30秒，问自己："我现在在感受什么？"
> 不需要做任何事，只需要**注意到**。

### 晚间（5分钟）
1. **身体扫描**（2分钟）——从头到脚感受身体
2. **情绪温度计**（1分钟）——"今天我感受到了____，强度____，发生在____"
3. **图式觉察**（2分钟）——"今天有没有图式被触发的时刻？我是怎么反应的？我想怎么反应？"

### 每周（30分钟）
> 回顾本周的情绪日记，找模式：
> - 哪个图式出现最多？
> - 什么场景最容易触发？
> - 我的反应和上周比有变化吗？

## 5.3 进度追踪

每月做一次快速自评（用同一个1-7评分），对比原始分数：

| 图式 | 基线 | 第1月 | 第2月 | 第3月 | 趋势 |
|------|------|-------|-------|-------|------|
| 情感抑制 | 5.33 | | | | |
| 苛刻标准 | 4.67 | | | | |
| 被遗弃 | 4.33 | | | | |
| 情感剥夺 | 4.33 | | | | |
| 不信任 | 4.0 | | | | |
| 缺陷感 | 4.0 | | | | |

**正常的进步速度**：2-3个月内核心图式下降0.5-1.0分。不要期待图式消失，目标是**从被图式控制变成能观察图式**。

## 5.4 什么时候需要专业帮助

以下任何一条出现，建议找图式治疗师或EFT治疗师：

- [ ] 你做情绪练习时出现强烈的恐慌或解离（感觉"不真实"）
- [ ] 你在渐进暴露中反复失败，且自我价值感在下降
- [ ] 你发现自己在用更极端的方式压抑情绪（酗酒、自伤、暴食等）
- [ ] 你的睡眠/食欲/日常功能受到明显影响
- [ ] 你做安全基地意象时什么画面都出不来，只有黑暗/空洞
- [ ] 你持续感到绝望或无意义感

**找治疗师的标准**：找有图式疗法（Schema Therapy）认证的治疗师，或做依恋为基础的治疗（EFT）。CBT也可以但对你来说可能不够深。

# 第六章：对你当前阶段的特别建议

## 6.1 关于分手

你的分手可能还没有被**真正处理**。情感抑制让你"分析通了"，但情绪层面可能还压着。

**建议**：不要急着进入下一段关系。先用情绪日记处理这段关系中的感受——不只是分析"为什么会分手"，而是感受"分手让我**感觉**到了什么"。

可能的感受：
- 被抛弃的恐惧（"果然会被离开"）
- 羞耻（"是不是我不够好"）
- 愤怒（如果你允许自己感受到的话）
- 悲伤（这可能是你最需要感受但最不想感受的）
- 解脱（这也是一种真实感受，不需要为此愧疚）

所有这些感受都是合理的。不需要评判。

## 6.2 关于"找人生支柱"

你21岁，在找人生支柱。但你的图式可能在干扰这个过程：

- **苛刻标准**告诉你："你必须找到那个'正确'的答案"
- **情感抑制**告诉你："不要被感性的东西迷惑"
- **缺陷感**告诉你："你可能根本找不到"

真正的身份探索不是通过**思考**完成的，而是通过**体验**。你需要先允许自己感受，才能知道自己真正想要什么。

## 6.3 你最大的风险

你的图式组合有一个特殊的风险：**用"自我分析"替代"自我改变"**。

你会做这个评估，会觉得"嗯，分析得挺准"，然后继续用理性分析一切——但情感层面什么都没变。

**检验标准**：如果你做完这个评估后只是"理解了"，但生活中没有任何**行为**上的改变，那情感抑制又一次成功地把你隔离了。

**最小的行动**：今晚就做一次情绪温度计。不需要分享给任何人，不需要"分析"，只是写下："今天我感受到了______。"
