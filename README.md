# psychology-of-meaning-research

> 一个基于循证心理学的自我探索与改变系统
> 不是鸡汤，不是玄学——每一个工具都有学术依据

## 这个项目是什么

这是一个**个人成长的工程化项目**。

它的核心假设：心理问题不是"想通了就好了"，而是需要**系统性的理解 + 结构化的练习 + 持续的追踪**。

基于三个理论支柱：
1. **图式疗法**（Schema Therapy）— 理解深层模式从哪来
2. **依恋理论**（Attachment Theory）— 理解关系中的行为模式
3. **接纳承诺疗法**（ACT）— 理解如何带着不确定性行动
